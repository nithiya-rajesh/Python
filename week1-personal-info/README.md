# 📝 Personal Information Manager

## 📖 Project Description
This is a beginner-friendly Python program that stores and displays personal information.  
It demonstrates the use of variables, user input/output, string formatting, validation, and enhancements like age calculation.

## ⚙️ Features
- Stores static information (name, age, city, hobby)
- Collects user input for favorite food and color
- Validates empty input (replaces with “Not provided”)
- Formats output using string methods (`.title()`, `.capitalize()`)
- Displays information in a nicely formatted block
- Calculates age in months
- Adds welcome and goodbye messages

## 🚀 How to Run
1. Clone the repository:
   ```bash
   git clone https://github.com/<your-username>/week1-personal-info.git
   ```
2. Navigate into the folder:
   ```bash
   cd week1-personal-info
   ```
3. Run the program:
   ```bash
   python personal_info.py
   ```
## 🧪 Testing
Use test_inputs.txt to simulate different inputs:

1. Normal input (valid food and color)

2. Empty input (food/color left blank)

3. Edge cases (long strings, unusual characters)
