# Week 4 Finance Tracker - Requirements

# No external dependencies required!
#

# - pathlib (built-in) - For cross-platform paths
# - datetime (built-in) - For date/time operations
# - collections (built-in) - For defaultdict
# - unittest (built-in) - For testing
#
# Installation:
# No pip install needed!
# Just ensure Python 3.7+ is installed
#
# To run:
